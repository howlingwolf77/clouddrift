1. *The ensemble's primary metric is AUC-ROC, not F1, because the temporal split produces a 1.1% validation anomaly rate that makes binary precision/recall targets mathematically unachievable regardless of model quality.*
2. *The ensemble (AUC-ROC=0.863) outperforms both individual models (IF=0.785, TCN=0.843), confirmed via a two-stage weight scan, demonstrating genuine complementary signal between point-wise feature-space detection and temporal sequence reconstruction.*
3. *Test-set AUC-ROC (~0.43) is lower than validation due to temporal non-stationarity — NAB's test period contains anomaly signatures not represented in the training period. This is a known limitation of single-split evaluation on heterogeneous real-world time-series and would be addressed in production via periodic retraining and drift monitoring (implemented in Phase 2D, Day 10).*
